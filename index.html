Flagship

Cosoul.AI

一套面向 AI 社交匹配场景的多智能体产品框架。用户可以创建多个 Persona，让 Agent 代表不同身份完成需求澄清、对象检索、协商撮合与连接确认。 项目采用 monorepo 组织 Web 与 Native 端，并在业务层接入 LLM、Embedding、记忆与任务状态机。
