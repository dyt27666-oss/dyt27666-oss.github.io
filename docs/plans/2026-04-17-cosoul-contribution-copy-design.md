# Cosoul Contribution Copy Design

日期：2026-04-17

## 目标

在不改动当前赛博朋克视觉结构的前提下，把首页 `Cosoul.AI` 主项目区从“系统介绍”强化为“团队主项目 + 我的角色 + 我的推进动作”的招聘表达。

## 问题

当前主项目区能够说明 `Cosoul.AI` 是核心项目，但对招聘方来说，个人贡献仍然偏模糊，容易被读成：

1. 项目本身很完整
2. 但候选人在项目中的具体价值边界不够清晰

这会削弱页面对 `AI工程师，兼具产品与全栈落地能力` 这一定位的支撑。

## 方案

采用保留视觉结构、重写主项目文案的方案：

1. 保留 `Cosoul.AI` 是团队主项目的表述，避免夸大个人贡献
2. 将主文案改为“主项目是什么 + 我在其中承担什么角色”
3. 将四个信号卡改写为更像招聘语言的“推进动作”

## 改写原则

1. 不写成“我独立完成了整个主项目”
2. 不写成纯技术堆栈说明
3. 强调跨角色推进：
   - 产品理解
   - Web / Native / Agent 协同落地
   - 围绕主项目抽离独立支撑系统
4. 让招聘方一眼看出这不是单点开发，而是系统推进能力

## 预期结果

改完后，`Cosoul.AI` 区域应该能同时回答这几个问题：

1. 这是个什么项目？
2. 你在里面到底做了什么？
3. 你为什么适合 AI 工程、AI 产品研发、复合型岗位？
