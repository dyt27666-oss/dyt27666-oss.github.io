# Selected Work Design

日期：2026-04-18

## 目标

在不改动现有 `Cosoul` 主线展示的前提下，把旧简历页中最有价值的补充内容接回当前主页，让页面更完整、更长，同时避免主线被冲散。

## 问题

当前主页围绕 `Cosoul` 项目族已经足够清晰，但也存在一个现实问题：

1. AI / Agent 主线很强
2. 但旧页面里能支撑数据分析、业务分析、研究建模基础的内容完全消失了

这会让页面更像“单一项目站”，而不是更完整的求职作品集。

## 方案

新增一个 `Selected Work` 补充区块，放在 `Cosoul` 主线和求职总结之后，作为能力补证。

只接回 3 组最有价值的内容：

1. `FitTrack AI`
2. `电商 GMV / RFM 分析`
3. `Research Highlights`

## 设计原则

1. 不改动现有 `Cosoul` 区块内容
2. 不让旧内容和 `Cosoul` 抢主线
3. 旧内容定位为“补足履历面”，不是“平级主项目”
4. 继续复用现有 gallery 组件，保证页面交互一致

## 预期结果

改完后页面会形成清晰结构：

1. `Cosoul` 作为主轴
2. `Selected Work` 作为补充证明
3. 招聘方既能看到 AI 工程主线，也能看到数据分析、产品化和研究基础
